# ViVaKR Template

```bash
dotnet new install Microsoft.TemplateEngine.Authoring.Templates
dotnet new templatepack -n "ViVaKR.Utility.Templates"

dotnet pack

# Windows pwsh
dotnet new install ./bin/Release/ViVaKR.Utility.Templates.1.0.0.nupkg
# Linux 또는 macOS에서:
dotnet new install bin/Release/ViVaKR.Utility.Templates.1.0.0.nupk

# uninstall
dotent new uninstall
```
